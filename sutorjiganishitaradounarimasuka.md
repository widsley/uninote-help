# ストレージが上限に達したらどうなりますか？

{% hint style="info" %}
ストレージ自動追加の設定によって、動作が異なります。デフォルトでは自動課金が行われます。
{% endhint %}

## A. 設定の状態によって、自動課金またはデータ書き込み停止のいずれかが発生します。

### 自動追加設定がONの場合（デフォルト）

* 容量の上限に達すると、不足分を補うためのストレージオプションが**自動的に購入・追加**されます。
* 追加は100GB単位（¥20,000/月）で行われます。

### 自動追加設定がOFFの場合

* 容量の上限に達した瞬間、データの追加・アップロードが**システム的にブロック**されます。
* 閲覧・ダウンロード・削除のみ可能な「読み取り専用（Read-Only）状態」になります。

***

### 事前のアラートについて

容量が以下の割合に達すると、段階的にアラートメールが送信されます。

* **80%** 到達時
* **90%** 到達時
* **95%** 到達時

また、自動課金が発生する **1週間前** と **前日** にも事前通知メールが届きます。
