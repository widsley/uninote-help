---
title: "Setting Up Formats"
---

<Note>
  In the editor, you can set the format, prompts, and custom fields for meeting summaries.
</Note>

---

### Things to Check in Advance

- Decide what type of summary you want to output (e.g., decisions, ToDos, risks, etc.).

---

### Steps: Open the Editor

1. Open the [Editor] screen.
2. Click on [Format] → [+ New Format].\
   <Frame>
     ![Image](/images/image-1.png)
   </Frame>
3. Enter the format name and click [Create].

---

### Create a Prompt

<Note>
  A prompt defines **how to structure the summary with blocks** for the created format. By writing instructions for each block, the AI will generate the summary in that format.
</Note>

1. Open [Prompt] and select the format you want to edit.
2. Choose how to use the template.
   - Use the default template as is
   - Add custom blocks based on the default template
   - Construct using only custom blocks without a template
3. Select the type of blocks to add.
   - **Plain Text**
   - **Bullet Points**
   - **Numbered List**
   - **Table with Borders**
4. Enter more detailed instructions in the box below the block.
   - Example: Please summarize the issues and needs gathered from the customer in bullet points.
   - Example: Please provide a summary of the entire meeting and the most important points.
5. Check the summary preview in [Preview].
   <Frame>
     ![Screenshot 2026 03 01 9 22 40](/images/スクリーンショット-2026-03-01-9.22.40.png)
   </Frame>
6. If everything looks good, click [Update] to finish.

<Note>
  The default template cannot be added to, deleted, or edited. Please select a template and edit the content as needed.
</Note>

---

### Select a Format in the Meeting

1. Open the calendar and click on [+ New Meeting] (or the relevant meeting).
2. Select the format you created in [Format] and update it.\
   <Frame>
     ![Screenshot 2026 06 05 17 18 47](/images/スクリーンショット-2026-06-05-17.18.47.png)
   </Frame>

---

### Set Up Custom Fields (Optional)

<Note>
  Custom fields are defined items for integration with external tools like Salesforce (e.g., project name, product, deal stage, etc.). This is a separate function from the AI summary blocks.
</Note>

1. Open [Custom Fields] in the editor.
2. Click on [+ New Field].
   <Frame>
     ![Screenshot 2026 06 05 17 20 50](/images/スクリーンショット-2026-06-05-17.20.50.png)
   </Frame>
3. Set the field name, field ID, and field type, and select the applicable format.
4. Click [Add Field] to finish.\
   <Frame>
     ![Screenshot 2026 06 05 17 22 39](/images/スクリーンショット-2026-06-05-17.22.39.png)
   </Frame>
