---
title: Can I set a password if I log in with a Google / Microsoft account?
---

<Tip>You can set a password. It is possible to set a new password using the password reset feature.</Tip>


## A. Yes, you can set a password using the password reset feature.

If you signed up with a Google or Microsoft account, a dedicated password for Uninote is not set by default.

### Steps to set a password

1. Click on [Forgot your password?] on the login screen.
2. Enter your registered email address.
3. Set your password using the link in the email you receive.

After setting it, you will also be able to log in with your email address and password.

<Note>You can continue to use login with your Google / Microsoft account.</Note>
