# アカウントの名前を変更するにはどうすればいいですか？

{% hint style="success" %}
アカウント設定からいつでも名前を変更できます。変更後は即時反映されます。
{% endhint %}

## A. アカウント設定からいつでも変更できます。

### 手順

1. 画面右上のアカウントアイコンをクリックします。
2.  ［マイアカウント］を開きます。<br>

    <figure><img src=".gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure>
3.  ［編集］をクリックし、保存します。<br>

    <figure><img src=".gitbook/assets/image (30).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
名前の変更はアカウント設定からいつでも行えます。変更後は即時反映されます。
{% endhint %}
